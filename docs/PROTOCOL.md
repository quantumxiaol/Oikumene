# Protocol

Protocol version: `0.1`

The protocol is advisory. C++ sends compressed strategic reports and legal candidate actions. Python returns suggested action IDs.

## GET /health

Checks whether PythonAgentServer is reachable.

Response:

```json
{
  "status": "ok",
  "service": "OikumeneAgentServer"
}
```

## POST /api/v1/decisions/batch

Returns suggested actions for one or more polities.

Request shape:

```json
{
  "protocol_version": "0.1",
  "requests": [
    {
      "protocol_version": "0.1",
      "request_id": "turn_10_polity_1",
      "turn": 10,
      "strategic_turn": 2,
      "polity_id": 1,
      "report": {
        "version": "0.1",
        "polity": {
          "id": 1,
          "name": "Barkaquin Confederacy",
          "population": 842,
          "settlement_count": 4
        },
        "stockpiles": {
          "food": 1240,
          "wood": 330,
          "ore": 42,
          "wealth": 19
        },
        "governance": {
          "stability": 0.82,
          "overextension": 0.14
        },
        "neighbors": [],
        "war_targets": [],
        "vassal_treaties": []
      },
      "candidate_actions": [
        {
          "action_id": "research_mining",
          "type": "RESEARCH_TECH",
          "target": "mining",
          "cost": 40,
          "estimated_benefit": 0.8,
          "risk": 0.1
        }
      ]
    }
  ]
}
```

Response shape:

```json
{
  "protocol_version": "0.1",
  "responses": [
    {
      "protocol_version": "0.1",
      "request_id": "turn_10_polity_1",
      "turn": 10,
      "polity_id": 1,
      "selected_actions": [
        {
          "action_id": "research_mining",
          "priority": 0.7,
          "commitment": "medium"
        }
      ],
      "diplomatic_messages": [],
      "rationale_short": "Selected the highest benefit candidate action.",
      "confidence": 0.55
    }
  ]
}
```

## StrategicReport

`report` is generated by C++ from authoritative simulation state. It is intentionally compressed for LLM use:

- It includes aggregate polity status, stockpiles, budget, governance, technology, geography, trade, diplomacy, wars, occupations, vassal treaties, and recent events.
- It does not include the full map, full route paths, all tiles, or unbounded event history.
- `candidate_actions` are the only legal action IDs Python may choose from.
- `oikumene_sim_batch` exports `strategic_reports.json` and `decision_batch.json` so the report shape can be reviewed before connecting a live Python/LLM provider.

## Validation Rules

C++ must reject any response that:

- Has the wrong request ID.
- Targets the wrong turn or polity.
- Selects an action ID not present in `candidate_actions`.
- Selects an action that is no longer legal.
- Is missing required protocol fields.
